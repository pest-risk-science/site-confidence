# site-confidence
Repo for Confidence in Pest Prevalence in a site.

This repo contains 3 main scripts: 
- main.R
- analysis.R
- figures.R

Simulations are first run using main.R, with aggregation done via analysis.R, and finally figures.R generating plots.  

This repo also depends on the package 'trapDetect' (https://github.com/pest-risk-science/trapDetect).  

Simulations used for the paper are found on CSIRO's data access portal (https://data.csiro.au/collection/csiro:69397v1)
